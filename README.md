# Web utils (WIP)

```sh
npm install @shedevro/web-utils --save
```

```javascript
const { Assert } = require('@shedevro/web-utils');
Assert.string(...);
...
```

```typescript
import { Assert } from '@shedevro/web-utils';
Assert.string(...);
...
```
